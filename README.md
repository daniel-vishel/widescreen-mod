# Dawn of War (Anniversary Edition) — Widescreen + Camera Zoom

Два независимых мода для Warhammer 40,000: Dawn of War — Anniversary Edition (Steam):

| Мод | Что делает | Трогает ли файлы игры | Откат |
|---|---|---|---|
| [widescreen](widescreen/) | Честное расширение обзора под широкий экран (не растяжение), UI стандартный | **Нет** (патч в памяти процесса) | Запустить игру обычным ярлыком |
| [camera-zoom](camera-zoom/) | Увеличенный отвод камеры колёсиком | **Нет** (новый файл поверх архива) | `-Restore` (удаляет один файл) |

Тестируется на **3440x1440 (21:9)**, поддерживаются любые разрешения.

> ⚠️ **Только для одиночной игры.** Изменённый aspect в мультиплеере — информационное
> преимущество и риск рассинхронизации.

Требования: Windows, PowerShell 5.1+ (есть в системе), игра установлена через Steam.
Если PowerShell ругается на политику запуска скриптов, запускайте так:
`powershell -ExecutionPolicy Bypass -File <скрипт.ps1>`

---

## 1. Widescreen (расширение обзора)

### Как это работает

Движок хранит соотношение сторон как константу `1.3333` (4:3, hex `AB AA AA 3F`)
в `Platform.dll`, `spDx9.dll`, `UserInterface.dll`. Замена на реальное соотношение
экрана (`3440/1440 = 2.3889`) заставляет движок рендерить **более широкую область
мира**: пропорции корректны, круги выделения юнитов остаются кругами.

Лаунчер `Start-DoWWidescreen.ps1` запускает игру и меняет эту константу
**в памяти процесса** — файлы игры на диске не изменяются. Единственная правка
на диске — разрешение в конфиге `Local.ini` (перед правкой создаётся копия
`Local.ini.wsbak`).

### Установка / запуск

```powershell
cd widescreen
.\Start-DoWWidescreen.ps1                      # 3440x1440 (по умолчанию)
```

Другие варианты:

```powershell
.\Start-DoWWidescreen.ps1 -Width 2560 -Height 1080   # другое разрешение
.\Start-DoWWidescreen.ps1 -Game WA                   # Winter Assault (W40kWA.exe)
.\Start-DoWWidescreen.ps1 -Attach                    # игра уже запущена — только патч
.\Start-DoWWidescreen.ps1 -GamePath "D:\...\Dawn of War Gold"  # если автопоиск не нашёл
```

Играть через лаунчер нужно **каждый раз** (патч живёт только в памяти запущенного
процесса) — это и есть «мод без установки». Удобно сделать ярлык:

```
powershell -ExecutionPolicy Bypass -File "<путь>\Start-DoWWidescreen.ps1"
```

### Мини-карта (известный трейдофф)

Константа в exe отвечает и за мини-карту. Режимы `-ExeMode` (патчится тоже только память):

| Режим | exe | Эффект |
|---|---|---|
| `skip` (по умолч.) | не трогается | мир корректный; мини-карта может быть искажена |
| `compromise` | 1.25 | частично чинит мини-карту; побочно искажает главное меню (в игре всё ок) |
| `full` | полный aspect | мини-карта ок у части пользователей, но мир может растянуться |

Порядок теста: `skip` → если мини-карта сломана → `compromise` → сравнить.

### Откат

- **Полный откат мода** — просто запустите игру обычным ярлыком Steam: без
  лаунчера игра полностью оригинальна.
- Вернуть исходный `Local.ini`: `.\Start-DoWWidescreen.ps1 -RestoreIni`

### Важно

- **Не меняйте разрешение в настройках игры** — игра перепишет `Local.ini`.
  Разрешение задаётся только лаунчером.
- Запасной вариант — дисковый патчер `widescreen\DoW-Widescreen-Patcher.ps1`
  (меняет файлы на диске, с полным бэкапом и откатом `-Restore`). Использовать,
  только если лаунчер не сработал (например, патч в памяти не успевает до
  инициализации рендера).

## 2. Camera Zoom (отвод камеры колёсиком)

### Как это работает

Параметры камеры лежат в `Engine\Engine.sga` в файле `camera_high.lua`.
Движок читает **loose-файлы из `<игра>\Engine\Data` поверх архива**, поэтому
установщик просто кладёт туда модифицированный `camera_high.lua`
(максимальный отвод `DistMax`: 38 → 76). Ни один оригинальный файл не изменяется.

### Совместимость с текущими кампаниями

В отличие от гайдов из Steam Community, где мод ставится как отдельный модуль
(и требует новой кампании), здесь файл кладётся на **engine-уровень**, а не в
данные модуля кампании (W40k/WXP). Параметры камеры не хранятся в сейвах —
**уже идущие кампании и сейвы продолжают работать**, никакого перерендера не нужно.

### Установка

```powershell
cd camera-zoom
.\Install-CameraZoom.ps1                 # DistMax = 76 (2x от оригинала)
.\Install-CameraZoom.ps1 -DistMax 100    # дальше (но больше тумана вдали)
.\Install-CameraZoom.ps1 -WheelSpeed 2.0 # быстрее зум за один щелчок колёсика (ориг. 1.45)
```

Ручная установка (без скрипта): скопировать `camera-zoom\ready\camera_high.lua`
в `<игра>\Engine\Data\` (папку `Data` создать, если нет).

### Обязательная настройка в игре

В настройках графики игры включите **Full 3D Camera** (полная 3D-камера) —
иначе используется упрощённая камера (`camera_low`) и мод не действует.

### Откат

```powershell
.\Install-CameraZoom.ps1 -Restore        # удаляет установленный файл
```

### Известные особенности

- На дальнем отводе карта затягивается туманом (особенность движка). Чем больше
  `DistMax`, тем заметнее. 76 — разумный баланс; подбирайте под себя.
- Мод действует на все режимы (скирмиш, кампании DoW и Winter Assault).

## 3. Чек-лист вечернего теста

1. **Widescreen, режим по умолчанию**: `.\Start-DoWWidescreen.ps1`
   - [ ] лаунчер отчитался `[OK]` по трём DLL;
   - [ ] в бою обзор шире (видно больше по бокам, не растянуто);
   - [ ] круги выделения юнитов круглые;
   - [ ] мини-карта: скриншот; если искажена → `-ExeMode compromise`, сравнить;
   - [ ] главное меню, ролики, edge scrolling у краёв экрана.
2. **Camera zoom**: `.\Install-CameraZoom.ps1`
   - [ ] Full 3D Camera включена в настройках;
   - [ ] колёсико отводит камеру заметно дальше;
   - [ ] загрузка существующего сейва кампании работает;
   - [ ] туман на максимальном отводе терпим (иначе уменьшить `-DistMax`).
3. Зафиксировать выбранный `-ExeMode` и `-DistMax` → обновить README.

## 4. Справка: hex-значения соотношений сторон

| Соотношение | Разрешение | Float | Hex (LE) |
|---|---|---|---|
| 4:3 (оригинал) | 1600x1200 | 1.3333 | `AB AA AA 3F` |
| 16:10 | 1920x1200 | 1.6000 | `CC CC CD 3F` |
| 16:9 | 1920x1080 | 1.7778 | `39 8E E3 3F` |
| 21:9 | 2560x1080 | 2.3704 | `26 B4 17 40` |
| **21:9** | **3440x1440** | **2.3889** | **`8E E3 18 40`** |
| 32:9 | 5120x1440 | 3.5556 | `39 8E 63 40` |
